# 🎯 Number Guessing Game

## Objective

The objective of this project is to create an interactive and engaging number guessing game where players attempt to guess a randomly generated number between 1 and 100. The game enhances logical thinking and problem-solving skills by providing hints and a scoring system based on the number of attempts used.

## Tools & Technologies Used

* React 18
* TypeScript
* Vite
* Tailwind CSS
* HTML5
* CSS3
* JavaScript (ES6+)
* VS Code
* GitHub

## Steps Performed

1. Designed a responsive and user-friendly game interface.
2. Implemented random number generation logic.
3. Added user input validation and smart hints.
4. Developed a scoring system based on attempts taken.
5. Created a statistics dashboard to track wins, losses, score, and win rate.
6. Added round history and guess history features.
7. Tested the application for functionality and responsiveness.
8. Uploaded the source code and documentation to GitHub.

## Outcome

The project successfully delivers a fun and interactive Number Guessing Game with modern UI design and responsive functionality. Players can enjoy multiple rounds, track their performance through detailed statistics, and improve their guessing strategies. The application demonstrates frontend development skills, state management, and responsive web design using modern technologies.
